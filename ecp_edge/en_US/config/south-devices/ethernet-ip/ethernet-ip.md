# EtherNet/IP

## 模块描述

此驱动主要用于支持 EtherNet/IP 协议的设备。

## 设备配置

| 字段 | 说明                  |
| ---- | --------------------- |
| host | 设备 IP 地址            |
| port | 设备端口，默认为44818 |
| slot | CPU 槽号，默认为0      |

## 支持的数据类型

* INT8
* UINT8
* INT16
* UINT16
* INT32
* UINT32
* INT64
* UINT64
* FLOAT
* DOUBLE
* BOOL
* BIT
* STRING
* WORD
* DWORD
* LWORD
